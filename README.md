Fungsi:
Menghitung hasil faktorial dari bilangan positif (n!).

Cara kerja:

Pengguna memasukkan angka n.

Program menggunakan perulangan untuk menghitung 1 × 2 × 3 × ... × n.

Hasil akhir ditampilkan sebagai faktorial dari n.
